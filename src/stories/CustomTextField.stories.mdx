# CustomTextField

The `TextField` component is a customizable input field that can be used in forms. It includes options for labels, placeholders, tooltips, suffixes, and more.

## Basic Example

```jsx
import React, { useState } from 'react';
import { CustomTextField } from '@defactor/ui-kit';

const YourComponent = () => {
  const [value, setValue] = useState('');

  const handleChange = (event) => {
    setValue(event.target.value);
  };

  return (
    <CustomTextField
      value={value}
      onChange={handleChange}
    />
  );
};

export default YourComponent;

```
