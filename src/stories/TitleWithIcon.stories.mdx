## Examples

Below is an example of how to use the `TitleWithIcon` component with default properties:

```jsx
import { TitleWithIcon, GlobalIcon } from '@defactor/ui-kit'

<TitleWithIcon 
  label="Sample Title"
  image={GlobalIcon}
/>
```
Below is an example of how to use the TitleWithIcon component with custom properties:

```jsx
import { TitleWithIcon, GlobalIcon } from '@defactor/ui-kit'

<TitleWithIcon 
  label="Custom Title"
  color="#7c7c7e"
  colorIcon="#7c7c7e"
  image={GlobalIcon}
/>

```
