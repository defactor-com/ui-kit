## TitleWithIcon

## Usage With Default Properties:

```jsx
import { TitleWithIcon, GlobalIcon } from "@defactor/ui-kit";

<TitleWithIcon label="Sample Title" image={GlobalIcon} />;
```

## Usage With Custom Properties:

```jsx
import { TitleWithIcon, GlobalIcon } from "@defactor/ui-kit";

<TitleWithIcon
  label="Custom Title"
  color="#7c7c7e"
  colorIcon="#7c7c7e"
  image={GlobalIcon}
/>;
```
