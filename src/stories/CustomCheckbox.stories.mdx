# Custom CheckBox

This component provides a custom checkbox that can be toggled.

## Basic Example

```jsx

    <CustomCheckBox
    />

```

## Props

- **checkState**: `boolean` - Whether the checkbox is checked. (dafault: false)
- **completedState**: `boolean` - Whether the task associated with the checkbox is completed. (dafault: false)
