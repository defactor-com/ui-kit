# Custom CheckBox

This component provides a custom checkbox that can be toggled. It has properties to control its checked state and indicate completion. Use the controls below to interact with the component.

## Basic Example

```jsx

    <CustomCheckBox
    />

```

## Props

- **checkState**: `boolean` - Whether the checkbox is checked. (required)
- **completedState**: `boolean` - Whether the task associated with the checkbox is completed.
