## Example

Below is an example that that demonstrates hiding the `CustomTooltip` component when the tooltip text is empty:

```jsx
import { CustomTooltip } from '@defactor/ui-kit'

<CustomTooltip 
 />
```

Below is an example of how to use the `CustomTooltip` component with default properties:

```jsx
import { CustomTooltip } from '@defactor/ui-kit'

<CustomTooltip
tooltipText="Custom tooltip text"
 />
```

Below is an example of how to use the CustomTooltip component with custom properties:

```jsx
import { CustomTooltip } from '@defactor/ui-kit'

<CustomTooltip 
  tooltipText="Custom tooltip text"
  iconColor="#4CAF50"
/>
```

Props

## Props

| Name          | Type     | Default                     | Description                              |
| ------------- | -------- | --------------------------- | ---------------------------------------- |
| `tooltipText` | `string` | `""` | Text displayed inside the tooltip.        |
| `iconColor`   | `string` | #A8B0B6   | Color of the tooltip icon. (Temporarily until the designer updates the palette.)     |
